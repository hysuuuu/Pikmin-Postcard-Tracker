# 皮克敏明信片紀錄

此工具提供一個簡易的平台，快速紀錄遊戲「Pikmin Bloom」中，明信片與好友的寄送紀錄。

# 使用方式

1. 進入頁面 https://hysuuuu.github.io/Pikmin-Postcard-Tracker/
2. 可以網頁版使用，或點選分享→加入主畫面（iOS），建立捷徑　（此兩種方式資料不共通）
3. 刪除主畫面捷徑即會重置資料。

## 功能介紹

### 管理好友清單

在欄位中輸入欲新增的好友名稱，新增完成後會顯示在下方列表，可點選好友名稱進行修改或刪除。

### 紀錄寄出明信片

在欄位中輸入欲新增的明信片名稱，並勾選寄出的好友，可點選名信面屏稱進行修改或刪除。

### 資料管理

#### 匯出 CSV

將資料匯出為 CSV 檔案。

#### 清除資料

將紀錄重置

#### 匯入 CSV

透過 CSV 檔案將資料匯入網頁
CSV 欄位需為：明信片、名稱、(可選)寄出日期

### 紀錄列表

以表格顯示紀錄之明信片、寄出之好友與寄出日期。

## 注意事項

此工具使用瀏覽器 local storage 儲存資料，可視為資料與瀏覽器綁定。若長時間未使用，資料可能會被清除，因此建議定期以「匯出資料」進行備份。
